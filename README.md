# Authentication System

Sistema de autenticação desenvolvido com HTML, CSS, JavaScript e Node.js, criado com o objetivo de praticar conceitos fundamentais de desenvolvimento Full Stack sem depender de geração automática de código.

> Este projeto faz parte da minha jornada de evolução como desenvolvedor, priorizando o entendimento dos conceitos antes da utilização de frameworks.

---

## 🚀 Tecnologias

### Front-end
- HTML5
- CSS3
- JavaScript (Vanilla)

### Back-end
- Node.js
- Express

---

## ✨ Funcionalidades

- ✅ Tela de login
- ✅ Validação de campos no Front-end
- ✅ Validação de credenciais no Back-end
- ✅ API REST utilizando Express
- ✅ Comunicação Front-end ↔ Back-end via Fetch API
- ✅ Tratamento de erros HTTP
- ✅ Toast para mensagens de erro
- ✅ Redirecionamento após autenticação
- ✅ Interface responsiva em desenvolvimento
