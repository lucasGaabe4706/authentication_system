# Authentication System

Projeto desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web

## Objetivo

Construir um sistema de autenticação do zero, evoluindo gradualmente cada etapa do processo.

## Tecnologias

* HTML
* CSS
* JavaScript
* Node.js (em desenvolvimento)

## Funcionalidades implementadas

* Formulário de login
* Captura do evento de submit
* Prevenção do recarregamento da página
* Captura dos valores dos inputs
* Validação de campos vazios

## Próximos passos

* Criar servidor com Express
* Implementar rota POST `/login`
* Enviar dados do front-end para o back-end
* Criar usuário mockado
* Validar credenciais
* Retornar resposta para o usuário

## Objetivo de aprendizado

Este projeto está sendo desenvolvido como exercício prático para fortalecer conhecimentos em JavaScript, HTTP, APIs, autenticação e fluxo completo entre front-end e back-end.
